# Search & filters

Find assets quickly with full-text search and the filter bar.

## Search

Click the search bar or press `S` to focus it. Type any word - Damask searches asset names, tags, and custom field values.

Results update as you type.

## Filters

The filter bar below the search input lets you narrow results by:

- **Project** - limit to a specific project
- **Tags** - show only assets with selected tags
- **File type** - image, video, PDF, etc.
- **Custom fields** - filter by any defined field value
- **Date uploaded** - a date range picker

Combine filters freely - they all apply together (AND logic).

## Keyboard shortcut

Press `Ctrl+K` to open the command palette, which also searches your assets and navigates to recent items.

## Sorting

Use the sort buttons in the top bar to order by:

- Date uploaded (newest first by default)
- Name (A–Z)
- File size
