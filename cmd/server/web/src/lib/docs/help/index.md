# User guide

Welcome to Damask DAM. This guide covers everything you need to manage your digital assets effectively.

## Sections

- [Getting started](help/getting-started) - First login, workspace setup, your first upload
- [Your account](help/account) - Edit your user profile
- [Projects & folders](help/projects) - Organize assets into projects and folder trees
- [Assets](help/assets) - Upload, rename, tag, add custom fields, download
- [Variants](help/variants) - Create and manage asset variants
- [Workflows](help/workflows) - Automate actions on assets with event-driven workflow rules
- [Sharing](help/sharing) - Share assets with clients via secure links
- [Search & filters](help/search) - Find assets fast with full-text search and filters
- [Exporting](help/exports) - Download ZIPs, export the activity log, and set up automated exports to SFTP or Google Drive
