# Variants

Variants are derived versions of an asset - different sizes, formats, or watermarked copies.

## What are variants?

A variant is a processed copy of an original asset. Common uses:

- Resized images for web delivery
- Watermarked proofs for client review
- Converted video formats

Variants never replace the original. You can always go back to the source file.

## Creating variants

1. Open an asset
2. Go to the **Variants** tab
3. Click **+ New variant** and choose a type
4. Damask enqueues a background job - the variant appears when processing completes

## Auto-variants

Your admin can configure auto-variant rules via the **Image Router** settings. These run automatically on upload for matching asset types.

## Promoting a variant

If a variant should replace the original (e.g. you've retouched it), click **Promote** on the variant. This swaps it in as the primary file.

## Manual upload

You can also upload a manually processed file as a variant using **Upload variant** on the asset detail page.
