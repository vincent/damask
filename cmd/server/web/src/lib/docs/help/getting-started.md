# Getting started

Get up and running with Damask in minutes.

## First login

Open your Damask instance URL in a browser. If registration is enabled, click **Register** to create your account. Otherwise, use the credentials your admin provided.

After logging in you land on the Library - the central view for all your digital assets.

## Workspace setup

A **workspace** is a shared environment for your team. On first login you can create a new workspace or accept an invitation to join an existing one.

To configure your workspace:

1. Open the sidebar and click **Settings**
2. Set a workspace name and upload a logo if desired
3. Invite team members via **Members → Invite**

## Your first upload

1. Click the **Upload** button in the top bar (or press `Ctrl+U`)
2. Select files from your computer - images, videos, PDFs, and more are supported
3. Files appear in the asset grid as they finish processing

Damask automatically generates thumbnails and extracts metadata.
